# CloneCars

Proyecto creado por mí desde cero.

## Tecnologías

- React 19
- Vite 8
- ESLint

## Estructura del proyecto

```
cloneCars/
├── public/
│   └── favicon.svg
├── src/
│   ├── assets/
│   ├── images/
│   │   ├── Quincy.png
│   │   ├── Naomi.png
│   │   └── Yaw.png
│   ├── App.css
│   ├── App.jsx
│   ├── index.css
│   └── main.jsx
├── index.html
├── eslint.config.js
├── vite.config.js
└── package.json
```

## Instalación

```bash
npm install
```

## Arrancar en local

```bash
npm run dev
```

## Build

```bash
npm run build
```
